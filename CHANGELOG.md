# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-08

### Added
- Initial release of RepoExplorer
- Browse and explore GitHub repositories
- Multi-repository support with URL management
- GitHub repository search functionality
- File browsing and content viewing
- Copy to clipboard functionality
- Export files as JSON or TXT
- Favorites system for quick access to files
- Collections for organizing related files
- Custom tags with color coding
- Bulk selection and batch operations
- History tracking for recently viewed files
- Real-time logging system with multiple log levels
- Tag-based filtering
- Advanced search and filtering capabilities
- Settings management for repositories and tags
- Modern gradient UI with smooth animations
- LocalStorage persistence for all data
- Responsive design optimized for Caido

### Features
- **Repository Management**: Add, remove, switch between multiple GitHub repos
- **Organization**: Favorites, Collections, Tags for file management
- **Bulk Actions**: Select and perform actions on multiple files
- **Tracking**: History of viewed files and real-time action logs
- **Search**: Repository search and file filtering
- **Export**: Save files as JSON or TXT formats
- **UI/UX**: Beautiful gradient design with animations
